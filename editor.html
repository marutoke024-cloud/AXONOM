      How to Use

      図形を描く

      矩形・多角形・L字ツールを選び、キャンバス上をドラッグ。
      自由多角形「自由」を選びクリックで頂点を追加。ダブルクリックかEnterで確定。
      形を変える図形をダブルクリック→白い頂点ハンドルをドラッグ。空クリックで終了。
      厚みを付ける図形を選択→右パネルの「厚み」に数値入力。3D視点で立体になる。

      選択・編集

      選択 / 移動クリックで選択、そのままドラッグで移動。
      複数選択Shift＋クリックで追加・解除。または左の「複数選択」をONにしてクリック。Ctrl+Aで階の全選択。
      図形の結合2つ以上選択→左の「接合 / 型抜き / 抽出 / 除外」。最初に選んだ図形が型抜きの土台になる。
      複製 / 削除Ctrl+Dで複製、Deleteで削除。左のボタンでも可。
      色・透明度・枠線図形を選択→右パネルで個別設定。スウォッチでテーマ色を即適用。

      視点・画面

      平面図⇄3D上部の「Plan / 3D」ボタン。プリセット（真上 / Isometric / Cabinet）もワンクリック。
      角度の微調整右パネルのX/Y/Zスライダーまたは数値入力。リアルタイムに反映。
      ズーム / パンホイールでズーム、何もない場所をドラッグ（またはSpace＋ドラッグ）で画面移動。「Fit」で全体表示。
      フロア（階層）右パネルで追加・表示切替・▲▼で並べ替え。3D視点では階ごとに積み上がる。

      アイコン・文字・出力

      DCアイコン左のパレットから選び、置きたい場所をクリック。選択して右下ハンドルか右パネルでサイズ変更。
      監視カメラ配置後に右パネルの「向き」スライダーで視野の扇形が回転。
      テキスト「文字」を選び配置場所をクリック→入力してEnter。ダブルクリックで再編集。
      PNG書き出し上部「PNG」→背景（透明/白/テーマ）と解像度を選んで書き出す。今見えている角度のまま出力される。
      保存自動保存。上部「Projects」で複数マップの管理とJSON入出力。

      図形を描く

      矩形・多角形・L字下のツールを選び、キャンバスを指でドラッグ。
      自由多角形「自由」を選びタップで頂点を追加。描き終えたら「選択」ツールをタップすると確定（ダブルタップでも可）。
      形を変える図形をダブルタップ→白い頂点ハンドルを指でドラッグ。何もない場所をタップで終了。

      選択・編集

      選択 / 移動図形をタップで選択、そのまま指でドラッグすると移動。
      複数選択下の「複数選択」をONにすると、タップした図形が次々と選択に追加される（もう一度タップで解除）。図形の結合はこの状態で「接合」などを押す。
      複製 / 削除図形を選択して下の「複製」「削除」ボタン。
      色や厚みの変更図形を選択→右上「Panel」を開いてプロパティを調整。

      画面の操作

      拡大・縮小2本指でピンチ。上部の「−／＋」ボタンでも可。
      画面を動かす何もない場所を指でドラッグ。迷子になったら「Fit」で全体表示。
      視点の切替上部の「Plan / 3D」やプリセットをタップ。細かい角度は「Panel」内のスライダーで。

      アイコン・文字・出力

      DCアイコン下のパレットを左右にスクロールして選び、置きたい場所をタップ。
      テキスト「文字」を選んで配置場所をタップ→入力。ダブルタップで再編集。
      PNG書き出し / 保存上部バーを左右にスクロールして「PNG」「Projects」へ。作業は自動保存される。
